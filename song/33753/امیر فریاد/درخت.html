درخت

امیر فریاد
زمان آهنگ: 2 دقیقه و 31 ثانیه
تاریخ: آذر 16، 1392

متن موسیقی:

یه درخت تنها وسط درد و غم ها // مردم یا زندم یا هنوزم رو حوام ؟
از خانواده تا دوستاش همه کردن تردش // چون // فصلاش پاییز بود و زرد بوده رنگ برگش
لای لاشخورا عینه جغدا تا صبح میزد زل به شب // ریشش محکم تو خاک بود و هر روز میشد گنده تر
آسمونو دوست نداشت // بارونارو توف میکرد
گریه میکرد سیر بمونه هر روز میشد گشنه تر // خودش بود صاف و ساده بدون دوز و دغلی
نتونست از پا درش بیاره زربه ی هیچ طبری // برگ سبز نداشت و بازم تو مشت گرفت هرچی خواست
پاییزم که میشد ... // بازم با همه یه فرقی داشت
تنها دوستش خدا بود // با جنگلش حال نمیکرد // جا نمیشد رو زمین و آسمونو پاره میکرد
کوچیک که بود دورو ورییاش همش پشتش خوندن // حالا بزرگ شده قد کشیده همه زیرش موندن
من همون درخته جنگله شهرم // یک تنه ی خشکه خشک و روی گونم شبنم
تنها لای لاشخورا و حیوونای وحشی // بی نیاز از پشتیبان و بارونای کشکی
جوری باهام بدن انگاری که خودشون من و نکاشتن // تنها شدم چون تو جیبم برگای سبز نداشتم
نهال دیروز امروز شده گنده لای اره ها // داستان همون فراره گرگه و شاخ شدن بره ها
پایین بودم و دیدم دیگه جایی نیستش بره ما // آسمون و جر دادم با زرب شعر و سر صدام
خیلی وقته ، زرد و آروم ، پاییزو چیزه خواستی نیست // ولی هنوزم میجنگم و صورت سرخه با سیلی
دنیا همینه ، دفاع میکنی و گلتو نزده پاس میدی // منمو رولم و دوتا خط و چنتا قرص آسپیرین
شعر و بغض و اشک و درد و داد و چنتا قافیه // خدا و عشق و دود و قهوه
همینا واسم کافیه
