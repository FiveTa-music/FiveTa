ورس 1 [ جی دال ]:
میدونی بلدم چطوری من // یه طوری ربط ، بدم کلماتو یهویی رد
بدن ملت و یهویی غش // کنن برام نه نمیخوری به اون بخوری ترک
میخوری چک میخوری فک بیخودی نزن چون
روم زوم نگاهش بیشتره الان // از تو میکنه ولی ، میاد میکنه برام
میدونه توپ تو دست ماست // سر 22 دارم میشم یه غول تو رپ فارس
آره ، دنبال دردسر نه درد سر ولی ، در به در دنبالمه // پس بگو خفه شن چون رد بدم گندشونم خوراکمه
هر روز آره ریسکه // زندگیم و قلمم هم هنوز آره سیکه
دافت سیریشه هی میپیچه بهم // درست مثل مامورا تو ایسته ریزه
چشاشو پوسته تنش درست مثل ییو سفیده // بازم بهم شده خیره
میخواد تنو بده بهم به جاش دلو بگیره // ولی به ک×رت
همین یکی که نیست فقط بیشترم میپرن روم ، انگاری جلوم یه تیمه

کروس [ جی دال ]:
مثل شیکر سفیده ولی تلخه مزش // انگاری زدی سم به لثت
میده باز مغز من رد / میرم رو خط بعد اَه / میگم از زندگی سگی میگم از خط به خط رپ ( ×2 )

ورس 2 [ لیتو ]:
کاغذم پره غلط غلوط // دختره اومد شلپ شلوپ
داد رفت بعدم فرود // گاه برنامه بعدم گروپ
سیخه برنامه علل الطلوع // هبلن ولی انن تو روم
میگه آهنگاتو حفظم ، ولی فن نیستن چرند دروغ
من نیستم اهل نصیحت // وقتی که ترکه تحصیلم
مغزه من رفته تسلیت // جوّگیرم تحت تاثیرم
دخترا پر رو بلدن // میپرن چون زود بغلم
هی میرن قربون صدقم // میمونن دو روز اقلن
خرابه وضعیتم // په نکن اذیتم
مریضه مغزیتم // داغون نی مصلیتن
میپاشه از ریم سم // از بس این لحظه ها سفیده
من که مزه ها چشیدم // من چه لحظه ها که دیدم
