Щ…ШӘЩҶ Щ…ЩҲШіЫҢЩӮЫҢ:

ЩҲШұШі 1 :
ШҜЩҲШЁШ§ШұЩҮ Ъ©Ш§ШәШ°Щ… Щ…ЫҢШҙЩҮ ЩҫШ§ШұЩҮ ЩҫШ§ШұЩҮ ШЁШ§ ЩӮЩ„Щ… / Ш§ЩҲЩҶЫҢ Ъ©ЩҮ ШҜШ§ШұЩҮ Щ…ЫҢШұЩҮ ШӘЩҲЫҢЫҢ ЫҢШ§ Щ…ЩҶЩ…
ШӘЩҲ ЪҜЩҒШӘЫҢ Щ…ЫҢЩ…ЩҲЩҶЫҢ Щ…Ш«Щ„ЩҮ ЩӮШЁЩ„ЫҢ ЩҶЩ…ЫҢШұЫҢ / ШӘЩҲ ЩҮЩ…ЩҲЩҶЫҢ ЩҲЩ„ЫҢ ШЁЩҮ Щ…ЩҶ ШӯШіЩ‘ ЩӮШЁЩ„ЫҢ ЩҶЩ…ЫҢШҜЫҢ
ШӘЩҲ ЪҜЩҗЩ„Щҗ Ъ©ШҙШӘЫҢЩ…ЩҲЩҶ ШҢ ШӘЩҮ ШұШ§ЩҮЫҢЩ… ЩӮШ·Ш№Ш§ / ЩӮЩ„Щ… ШЁЩҮ ЫҢШ§ШҜШӘ Щ…ЫҢЪҜЫҢШұЩҮ ШұШ№ШҙЩҮ ЪҜШ§ЩҮЫҢ Ш§Щ„ШЁШӘ
ШәЩ… ШЁЩҮ ЫҢШ§ШҜШӘ Ш§ШәЩ„ШЁ Щ…ЫҢШҙЩҮ ШұЩҮШіЩҫШ§Шұ ЩӮЩ„ШЁЩ… / ЫҢШ§ШҜЪҜШ§ШұЫҢШӘ ЩҮШіШӘ ШұЩҲЫҢ ШұЩӮШөЩҗ ЩҫШ§ЫҢ ЩҶШЁШ¶Щ…
ШЁШ§ Ш§ЩҲЩҶЩ… ШӘЩҲ ШЁШ§ШұЩҲЩҶ Щ…ЫҢШұЫҢ ШІЫҢШұ ШўЩ„Ш§ЪҶЫҢЩӮ / ШіШұШӘЩҲ Щ…ЫҢШІШ§ШұЫҢ ШұЩҲ ЩҫШ§Шҙ Ш®ЫҢШұЩҮ ШЁШ§Щ„Ш§ ШҙЫҢ
ШЁШ§ Ш§ЩҲЩҶЩ… Щ…Ш«Щ„ЩҮ Щ…ЩҶ ЩҒЪ©Шұ Щ…ЫҢЪ©ЩҶЫҢ ШЁЩҮ ШҜЩҲШұ ШҜШіШӘ / ЫҢШ§ Ъ©ЩҮ ЩҶЩҮ ШЁЩҮ Ш§ЩҲЩҶЩ… Щ…ЫҢШҜЫҢ ШЁЩҮ ШІЩҲШұ ШҜШіШӘ
ЩҶЩ…ЫҢШ®ЩҲШ§Щ…ШӘ ШҢ ШӘЩҲ ЫҢЩҮ ШӯЫҢЩ„ЩҮ ЪҜШұЫҢ / ЩҶЩҒШұШӘ Ш№ШҙЩӮЫҢ Ъ©ЩҮ Ъ©ШұШҜЩҮ ШЁЩҲШҜ ЩҫЫҢЩ„ЩҮ ШҜШұЫҢШҜ
ЩҮЩҶЩҲШІ ЫҢЩҮ ЪҶЫҢШІЫҢ ЩҮШіШӘШҙ ШӘЩҲ Щ…ШәШІЩ… ЪҜЩҸЩҶЪҜ / ЫҢШ№ЩҶЫҢ ШЁШ§ШІЫҢ Щ…ЫҢЪ©ШұШҜЫҢ ЩҲШ§ШіЩ… ШұЩҸЩ„

ЩҮЩ…Ш®ЩҲШ§ЩҶ :
Ш®ЩҲШ§ШіШӘЩ… Ъ©ЩҮ ШұШ§ШӯШӘ ШӘШұ ШЁЪҜЩ… /ЪҶЩ…ШҙЩ…Ш§ШӘ Щ…ЫҢЪҜЩҒШӘ Ъ©ЩҮ ШЁШ§ Щ…ЩҶ ШӯШұЩҒ ШЁШІЩҶ
ЪҜШ°Ш§ШҙШӘЫҢ ЩҮЩ…ЩҮ ШЁШ§ Щ…ЩҶ ШЁШҜ ШЁШҙЩҶ / ШӘЩҲ ШҙШҜЫҢ ЩҒШұШҙШӘЩҮ ШӘШ§ Щ…ЩҶ ШЁШҜ ШЁШҙЩ…

ЩҲШұШі 2 :
Щ…ЩҶ ШЁШ§ ШӘЩҲ ШұЩҲ ШЁЩҲШҜЩ… ШҢ ШӘЩҲ ШұЩҒШӘЫҢ ШІЫҢШұ ШўШЁЫҢ / ШҙЩҶЫҢШҜЩ… ШҙШЁШ§ ШЁШ§ Ъ©Ші ШҜЫҢЪҜЩҮ Ш§ЫҢ Щ…ЫҢШ®ЩҲШ§ШЁЫҢ
ЪҶШ·ЩҲШұ ШЁШ§ШІЫҢ ШҜШ§ШҜЫҢ ШҜЩ„ЩҲ ЩҮШұ Ш·ШұЩҒ Ъ©ЩҮ Ш®ЩҲШ§ШіШӘЫҢ / ЪҜШ°Ш§ШҙШӘЫҢ ШұЩҒШӘЫҢ ШӘШ§ Щ…Ш¶ШӯЪ©ЩҮ ЫҢ Ш№Ш§Щ… ШҙЫҢЩ…
ШЁШұЩҲ ШЁЫҢШұЩҲЩҶ Ш§ШІ ШІЩҶШҜЪҜЫҢЩ… Ш§ШІ Ш°ЩҮЩҶЩ… / ШЁШҜЩҲЩҶ ШЁЫҢ ШӘЩҲ ЩҶЩ…ЫҢЩ…ЫҢШұЩ… ШҢ Щ…ЩҶ ШІЩҶШҜЩ…
ЩҮЫҢЪҶ ЩҲЩӮШӘ ЩҶШҙШҜ ШЁЩҮ Щ…ЩҶ ШЁШҙЩҮ Ш№ШҙЩӮШӘ Ш«Ш§ШЁШӘ / ШІЫҢШ§ШҜЫҢ ШЁШ§Щ„Ш§ ШұЩҒШӘЩҮ ШЁЩҲШҜ Ш§ШІ Щ…ЩҶ Ш§ЩҶШӘШёШ§ШұШӘ
ШӘЩҲ ШЁЩҮ Щ…Ш§ ЩҶЩ…ЫҢШ®ЩҲШұЩҮ ШӘЫҢЩҫШӘ Ъ©Ш§ШұШӘ Ш°Ш§ШӘШӘ / ЪҜЩҒШӘЫҢ Ш§ЫҢЩҶШ§ШұЩҲ ШұЩҒШӘЫҢ ЩҫЫҢ Ш№ШҙЩӮЩҸ ШӯШ§Щ„ШӘ
Ш¬ШҜЫҢШҜЩҮ ЪҶЩҫШҙ ЩҫШұЩҮ ШҢ ШҜШ§ШұЩҮ ЩҫЩҲЩ„ ШӘЩҲ Ш¬ЫҢШЁШҙ / ШұШ§ ШЁЩҮ ШұШ§ ШЁШұШ§ШӘ Щ…ЫҢШ®ШұЩҮ ЪҜЩ„ЩҸ ШұЪҳЩҸ ШұЫҢЩ…Щ„
Ш§Щ…ЫҢШҜ ЩҲШ§ШұЩ… ШЁШ§ШҙЫҢ ШҙШ§ШҜ ШҙШ§ШҜ / Ш§ЩҲЩҶ ШҜЩ„ Щ„Ш№ЩҶШӘЫҢШӘ ШҜЩ„Щ…ЩҲ ШЁШ§ШІЫҢ ШҜШ§ШҜ
Ш®Ш§Ш·ШұШ§ШӘШӘ ШӘЩҲ Ш°ЩҮЩҶЩ… ШіШұЪҜШұШҜЩҲЩҶЩҮ / ШҜЫҢЪҜЩҮ ЩҶЩ…ЫҢШ®ЩҲШ§Щ… Ш®ШҜШ§ ШӘЩҲШұЩҲ ШЁШұЪҜШұШҜЩҲЩҶЩҮ
