ورس 1

دوباره صدای ساعت روی دیوار // یه قلمه خسته با یه ذهنه بیدار
یه کاغذ خالی واسه شروع قصمون // یه حس مرده با یه بغز تو گلوم
که همین کافیه خب باز تا دم صبح // نبش قبر کنمو باز خاطراتو
تا که ببینم آخه چرا همیشه بن بست // رو به رومه تو جسمم یه روحه مردس
سقفه آرزوهامم همش که ترک // مرده بودو رفت اونم به درک
بزا عمر بگذره بزا ثانیه رد شه // حرفام قصه نیست واسه لالایی بچه
آره بد بختیا منو دوره کردن // من همون دیوونه ی دورگردم
که الان گذشته آب از سرم // یه اخلاقه گند از سگ بد ترم
میدونی چرا؟ چون بد بختیا فقط واسه ما بود // ما شدیم پای لنگو سنگ جلو پامون
که هرکس به سمتم از بخته بد // خنجر زدش که از پشت به من
آره من وصلم به دردم // همه زخمام موندن بی مرهم
من یه کوهم باشم بازم من // تو این بازی بازم بازندم

ورس دو

این قصه تلخه اره خیلی وقته خب واقعیت همینه // روز خوش بود مگه چشمام نادیده بگیره
الانم که میبینی بدجوری خستم // از آدمای دورم که از خودم پست تر
که با منو روزگارم یه جوری وصلن // بی انصاف نه نبودم اصلا
منی که ندیدم خوشی تو گذر عمرم // من حتی از عشقمم رو دست خوردم
حالا چی مونده برام از رفیق بگم ؟ // نداشتش مرام یا که از مردم بگم
که مارو پس زدن // به زیرزمین بودنمون انگ بد زدن
ما که نخواستیم شما به ما بالو پر بدین // اصلا نخواستیم یه کوچولو مارو درک کنین
ولی چیزی که ندیدی به چشم تهمت نزن // به یه مشت حرف مفت دامن نزن
ما که گفتیم 100 بار شکستیم // از اول همین بودیمو هستیم
فرقی نکردیم بازم همینیم // از همه خوردیم که الان نقش زمینیم
پس تو بگو انتقام از کی بگیرم ؟ // منی که یه عمره با گریه خوابیدم
به عشق اونایی که تهمت زدن // قاضی حکم نداد گردن زدن
شاید راست میگنو اصلا من بدم // تو خوب باشو بیا بگذر ازم
نه واسه دردام مرهم نباش // فقط نمک رو زخمم نپاش
