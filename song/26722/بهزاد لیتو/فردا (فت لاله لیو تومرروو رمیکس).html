بهزاد لیتو
تاریخ: آبان 29، 1390

متن موسیقی:

ورس 1
روی تخت نشستی چشمات خماره // روی لب و گردنت هی ماچ دوباره
باختن دستات انگار قماره // لبات گرم و مرطوب انگار شماله
نبینم که هی تو غُر بزنی // وقتی میتونی با " لیتو " دور زمین دور بزنی
مریضیم و ما قرص همیم // " بهزاد " خوبه ولی یه کم تخسه همین
چرا دسته داغه داره می لرزه // اگه دور باشیم ولی بازم می ارزه
امشب دارم میرم ولی عید اینجام // مثه تو رو پیدا نمی کنم هیچ جا
نمیخوام برم ازت دور شم ولی مجبورم // دیر میگذره بدون تو شب روزم
به تو عادت دارم تو که شدی همخونم // ولی باید برم ترک کنم من خونم

کروس
‏I know we could live tomorrow
‏But I know I live today,
‏I know we could live tomorrow
‏But I don’t think we should wait! No

ورس 2
زندگی حکم‌ کردیم با کارتا‌ کت شراب خنک توی آنتروکت پالتو کت رسیدی تو شاردوگل میام دنبالت من با چهارتا گل
خیلی خوبیم با هم ، جلو دوربین بازم // مثله ما نیست حتی تو فیلما هم
می خوایم با هم باشیم // صبحا راحت پاشیم
نه که داور باشیم بریم دنبال بقیه // ما دوتا با همیم کل دنیامون غریبه
روی تختیم و دیگه ملافه خنک // آره زندگی سیگاره و تنها به تو نخ داده
هی به خودم میگم چرا به تو لم داده // شبا دختر بدی و روزا دختر خوب
وقتی رو پاهات خوابم نمیخوام پر قو // میخوام ساعت وایسه اینقدِ تند نره زود

کروس

‏But I know I live today,
‏I know we could live tomorrow
‏But I don’t think we should wait! No

ورس 3
نمیخوام صبر کنم تا که فردا بشه // با دست نشون بدن بگن هرجا چشه
یادته لب دریا دستاا به شن // هاا هاا ،‌ نه نمی خوام فردا بشه
میخوام برم به قلب // میخوام بگم که من // به تو گره خوردم مثه یه بند کفش
آره ، سُر بخوریم مثه یه قطره اشک // آروم من و تو با هم بریم به تش
بذار در بیارم برات گوشواره هات // چون که باهم میخوایم بریم زیر دوش آب داغ
" بهزاد " شده عاشق بوت آره باز // نمیدونم دورت ملافه بود یا لباس
ما با هم تو عکسیم و منظره ها // عوض میشن آدما میان سر را
ولی مهم نیستن چون من تو رو دارم // میدونم دوست دارم تو رو واقعا
