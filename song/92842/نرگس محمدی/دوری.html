دوری

نرگس محمدی
زمان آهنگ: 3 دقیقه و 9 ثانیه
تاریخ: فروردین 10، 1404

متن موسیقی:

مه جا دوری و نویمبه تره یار
(از من دوری و ترو نمی بینم ای یار من )
چی تو ند آروم هاکنه مره یار (چی می تونه منو آروم کنه ای یار )
تره خامبه ولی دومبه نمونده (ترا می خوام ولی می دونم نمی مونه )
ت جا م وسه غیر خاطره یار
(از تو برام فقط خاطره می مونه )
امه میونه در فاصله (بین منو تو فاصله وجود دار)
کنه پلی من هاکنم گله (پیش کی گلایه کنم )
دیگر ندارمه حال و‌حوصله (دیگه حال و حوصله ندارم )
نونه نونه (نمیشه نمیشه )
مه دل دله غم دره دیا نشونه تا آروم بهیره (در دل من غم وجود دار و درون و بیرون دلم بسته شده بیرون نمیره غم تا قلبم آروم بگیره )
دیگه نتومبه دگردم شه یار بال سر بهمیرم (دیگه نمی تونم برگردم و رو دست یارم بمیرم )
چتی ون سر هاکنم روز و شور
(چطور باید من سر کنم روز و شب رو )
کدیم دریو بشندم شه چش او ر(کدوم دریا اشک چشمم بریزم )
آخه هیچی دل درد چاره نیه (آخه چیزی چاره درد دل نیست )
هیچ کس مه واری ون دل پاره نیه
( هیچ کس مثل من دلش پاره نیست )
نونه نونه نونه منه شو دیگه روز نونه (نمیشه نمیشه نمیشه شب من دیگه روز نمیشه)
جان خدا مه بخت چسه کم بهیتی (ای خدا جان چرا بخت منم کم گرفتی )
مه دور چه قلم بهیتی (دور منو چرا قلم گرفتی )
جان خدا هارش مه دل ر چه نخار( ای خدا جان ببین دل منو چقد بی حال )
نتونده درد تاب بیاره ( نمی تونه درد تحمل کنه )
اسا دیگه بیمه خسته (الان دیگه خسته شدم )
دل دیا راه دوسه (درون و بیرون دلم بسته است )
کجه بورم چتی بورم نونه نونه نونه یار (کجا برم چطور برم نمیشه نمیشه نمیشه ایریار)
منه شو دیگه روز نونه یار
(شب من دیگه روز نمیشه ای یار )
